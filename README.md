# 💧 Neer Nigrani - Water Monitoring System

## 📌 Project Description
Neer Nigrani is a web-based water monitoring system developed using PHP and MySQL.  
It helps in tracking water levels, managing complaints, and providing alerts for water shortages.

---

## 🚀 Features
- User Registration & Login System
- Forgot Password Functionality
- Water Level Monitoring
- Complaint Submission System
- Admin Dashboard
- Data Management using MySQL
- Simple & User-Friendly Interface

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: XAMPP / Localhost
